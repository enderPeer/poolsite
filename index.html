Das Konzept

Nutzer bauen das Netzwerk auf — und besitzen es

Klassische soziale Netzwerke verdienen mit den Inhalten ihrer Nutzer Milliarden — die Nutzer gehen leer aus. PoolSite dreht das Modell um: Die Anteile der Betreibergesellschaft (UG) werden über tokenize.it tokenisiert und Tag für Tag an die Community ausgeschüttet.

1

Aktiv sein & Punkte sammeln

Beiträge, Interaktionen und Engagement im Netzwerk werden über ein transparentes Punktesystem bewertet.

2

Täglich Token erhalten

Jeden Tag wird ein festes Token-Kontingent anteilig nach Punkten an alle aktiven Nutzer verteilt — im ersten Jahr 5.000 Token pro Tag.

3

Miteigentümer werden

Jeder Token repräsentiert einen echten Anteil an der Gesellschaft. Token-Halter partizipieren an Gewinnausschüttungen.

Tokenomics

18,25 Millionen Token. Mathematisch begrenzt.

Die Ausschüttung startet bei 5.000 Token pro Tag und sinkt jedes Jahr um 10 %. Das ist kein Zufall, sondern präzise konstruiert: Die Gesamtsumme aller jemals ausgeschütteten Token konvergiert exakt gegen 18,25 Millionen — 100 % der Anteile.

Warum genau 18,25 Millionen?
5.000 Token × 365 Tage = 1.825.000 Token im ersten Jahr. Mit −10 % pro Jahr ergibt sich eine geometrische Reihe: 1.825.000 × (1 + 0,9 + 0,81 + …) = 1.825.000 × 10 = 18.250.000. Der Grenzwert der Ausschüttung entspricht exakt der Gesamtzahl der Anteile — es kann nie mehr Token geben, als Anteile existieren.
Erlösmodell

Echte Einnahmen. Echte Ausschüttungen.

PoolSite finanziert sich über Werbung — direkte, planbare Einkünfte. Die Gewinne fließen vollständig zurück an die Token-Halter: teils über Buybacks, teils als Dividende.

① Werbeeinnahmen

Werbetreibende zahlen für Reichweite im Netzwerk — die direkte Einnahmequelle der Gesellschaft.

② Unternehmensgewinn

Nach Abzug der Betriebskosten entsteht der ausschüttungsfähige Gewinn.

③ Token-Buyback

Ein Teil der Gewinne kauft Token vom Markt zurück — das reduziert das Angebot und stützt den Wert.

④ Dividende

Der Rest wird als Dividende an alle Token-Halter ausgeschüttet — anteilig zum Bestand.
